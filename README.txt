我相信各位安裝xampp時一定沒有更改路徑對不對
請照以下方式執行

1. 解壓縮丟進 C:\xampp\htdocs
   index.php的路徑應為 C:\xampp\htdocs\db_topic1\index.php

2. 打開xampp，把 Apache 和 MySQL start

3. 把 C:\xampp\htdocs\db_topic1\db 裡的.sql檔匯入你的sql

4. 基本上打開 index.php 就會進到登入畫面了，我給的.sql裡面已經有一筆
   帳號：D1149754 密碼：123456 的資料，
   其他資訊直接在 phpMyAdmin 找了

其他 php 我都丟在 C:\xampp\htdocs\db_topic1\php裡了
